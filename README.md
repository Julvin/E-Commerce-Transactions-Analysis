# E-Commerce-Transactions-Analysis

By analyzing e-commerce platform transactions, we can infer platform's general performance and consumer behavior. These informations are crucial as reference points the business. By leveraging data analytics and visualization, we have generated the necessary informations that are required to aid platform's business decisions

Impact:
1. Identifying platform's key pain points 
2. Identifying platform's potential opportunity that could be capitalize
3. Reducing workhours on insight generations by creating data visualization in Power BI
